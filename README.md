# Python Quick start

> This is a python app example. 
> It 100% uses docker to do: development, test and build.

# Prerequisition
* docker
* docker-compose

**No need to install python!**

# Quick Start

## Test
```bash
make test
```
## Run
```bash
make run
make run ARGS="--version"
```
## Dist
```bash
make dist
```
