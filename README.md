# Python Quick start

> This is a python app example. 
> It 100% uses docker to do: development, test and build.

# Prerequisition
* docker: ">= 17.06"
* docker-compose: ">= 1.26"

# Quick Start

## Test
```bash
make test
```
## Run
```bash
make run
make run ARGS="-h"
```
## Dist
```bash
make dist
```
